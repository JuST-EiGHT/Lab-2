### Задание 1

#### 1.1 Постановка задачи
> Программа получает на вход 2 целых числа: начальное значение `a` и количество элементов `n`. Нужно вывести `n` элементов, элемент это сумма квадратов цифр предыдущего числа (для первого числа преодыдущее - это `a`).

#### 1.2 Входные и выходные данные
**Входные данные:**
`a` - Целое число (нулевой член), `n` - Натуральное, 0 < n < 2 147 483 647 (число следующих членов).

**Выходные данные:**
Последовательность из `n` целых чисел

#### 1.3 Выбор структуры данных
Программа получает 2 числа `a`, `n`. Также в программе присцтствет переменная для временного хранения результата `x` - int Поэтому для их хранения
можно выделить 4 переменных (`x`, `a`, `b` и `c`) типа `int`.

|             | название переменной | Тип (в Java) | 
|-------------|---------------------|--------------|
| X (Число 1) | `x`                 | `int`     |
| A (Число 2) | `a`                 | `int`     | 
| B (Число 3) | `b`                 | `int`     |
| C (Число 4) | `c`                 | `int`     | 

Для вывода результата необязательно его хранить в отдельной переменной.
Для хранения текущего элемента последовательности используем переменную типа `int`
Для счетчика цикла используем переменную типа `int`
Для временного хранения числа при вычислении суммы квадратов цифр используем переменную типа `int`
Для хранения текущей цифры числа используем переменную типа `int`
Для хранения суммы квадратов цифр используем переменную типа `int`

#### 1.4 Алгоритм
1. Считать начальное значение `a` и количество элементов `n`
2. Если `n <= 0`, вывести "NO" и завершить программу
3. Установить текущий элемент `x = a`
4. Повторить `n` раз:
   - Вывести текущий элемент `x`
   - Вычислить сумму квадратов цифр числа `x`:
     - Инициализировать сумму `sum = 0`
     - Скопировать значение `x` во временную переменную `temp`
     - Пока `temp > 0`:
       - Получить последнюю цифру: `digit = temp % 10`
       - Добавить квадрат цифры к сумме: `sum += digit * digit`
       - Удалить последнюю цифру: `temp = temp / 10`
   - Присвоить `x = sum`

#### 1.5 Программа
```java
import java.io.PrintStream;
import java.util.Scanner;

public class Main {
    public static Scanner in = new Scanner(System.in);
    public static PrintStream out = System.out;

    public static void main(String[] args) {
        // Считываем начальное значение и количество элементов
        int a = in.nextInt();
        int n = in.nextInt();
        
        // Проверяем корректность введенного количества элементов
        if (n <= 0) {
            out.println("NO");
            return;
        }
        
        // Инициализируем текущий элемент
        int x = a;
        
        // Генерируем и выводим последовательность
        for (int i = 0; i < n; i++) {
            // Выводим текущий элемент
            out.println(x);
            
            // Вычисляем сумму квадратов цифр текущего элемента
            int sum = 0;
            int temp = x;
            
            while (temp > 0) {
                int digit = temp % 10;
                sum += digit * digit;
                temp = temp / 10;
            }
            
            // Обновляем текущий элемент для следующей итерации
            x = sum;
        }
    }
}
```
